SMILES_to_IUPAC 
===================

Library for converting SMILES Presentation in IUPAC

Install
=======

Only python 3.8+.

* Install SMILES_to_IUPAC library DEV version for features that are not well tested::

    pip install -U git+https://github.com/walderhu/SMILES_to_IUPAC@develop
